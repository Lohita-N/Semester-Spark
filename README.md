# 🎓 Semester Spark

> A premium, startup-grade student productivity web app built to track your grades, simulate your future, and figure out exactly *how cooked* you are this semester. 

No more soul-crushing spreadsheets. Built with a gorgeous glassmorphism UI, cozy aesthetics, and gamified features to turn academic stress into motivation. 🚀

![Status](https://img.shields.io/badge/Semester%20Status-Locked%20In-emerald?style=for-the-badge)

---

## ✨ Core Features

* 📊 **Vibrant Dashboard:** Live GPA tracking, study hour rings, and dynamic motivation states (from *"Locked In"* 🔥 to *"Cooked"* 💀).
* 🧮 **Smart GPA Predictor:** Real-time cumulative calculations and a *"Final Exam Score Needed"* slider simulation.
* 📋 **Impact Kanban Board:** Manage tasks with a built-in *GPA Impact Estimator* so you know where to focus your energy.
* 📈 **Burnout Analytics:** Beautiful fluid charts mapping out your stress levels, study-to-grade correlation, and workload.
* 🍳 **"How Cooked Am I?" Engine:** A witty, AI-style reality check that gives you a survival percentage, dynamic roasts, and meme reactions.
* 🎨 **Cozy Themes:** Switch instantly between *Soft Academia* ☕, *Cyberpunk Finals* ⚡, or *Cozy Night* 🌙.

---

## 🛠️ Tech Stack

* **Core:** React ⚛️
* **Styling:** Tailwind CSS (Glassmorphism UI) 🌪️
* **Animations:** Framer Motion 🎬
* **Charts:** Recharts 📊
* **Icons:** Lucide React 🎨
